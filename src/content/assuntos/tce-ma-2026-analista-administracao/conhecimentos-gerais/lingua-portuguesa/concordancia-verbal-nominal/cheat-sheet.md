# Cheat sheet — Concordância verbal e nominal

## 1. Rastreie o controlador

> forma flexionada → função sintática → termo controlador → núcleo

- Não concorde por proximidade.
- Retire termos preposicionados e incisos.
- Em reescrita, propague gênero, número e pessoa por toda a cadeia.

## 2. Concordância verbal básica

| Estrutura | Forma segura |
| --- | --- |
| sujeito simples | verbo com o núcleo |
| sujeito composto anteposto | plural |
| sujeito composto posposto | plural ou atração |
| sujeito oracional | singular |
| infinitivos como unidade | singular |
| infinitivos como ações distintas | plural |

## 3. Sujeito composto

- **eu + outro** → 1ª pessoa plural.
- **tu + terceiro** → 2ª plural na tradição normativa.
- **ou exclusivo** → singular.
- **ou inclusivo** → plural.
- **nem... nem...** → plural preferencial.
- **um ou outro** → singular.
- **um e outro** → singular ou plural.
- **nem um nem outro** → singular editorialmente seguro.
- **com** equivalente a *e* → plural.
- termo com valor parentético → concordância com o primeiro núcleo.

## 4. Unidade de ideia

- “Ler e escrever **exige** concentração.”
- “Aprovar e executar **são** etapas distintas.”
- Sinônimos e gradação podem formar unidade no singular.
- Enumeração resumida: “Prazos, custos, riscos, **tudo foi** examinado.”

## 5. Partitivas e quantidades

- maioria de + plural → singular ou plural.
- coletivo simples → singular.
- **cada um dos**, **nenhum dos**, **qualquer um dos** → singular.
- mais de um → singular; reciprocidade ou repetição pode levar ao plural.
- fração → tendência de concordância com o numerador.
- percentual sem especificador → verbo com o número.
- percentual com especificador → examine percentual, nome e foco.
- um milhão → singular; dois milhões → plural.
- um total de cem processos → singular com *total*.
- cem processos → plural.

## 6. Relativos e tratamentos

- **que** sujeito → verbo com o antecedente.
- **quem** → 3ª singular ou antecedente.
- **um dos que** → plural preferencial.
- **qual de nós** → singular.
- **quais de nós** → 3ª plural ou concordância com *nós*.
- pronome de tratamento → verbo na 3ª pessoa; adjetivo conforme o sexo da pessoa.

## 7. Nomes e títulos plurais

- artigo plural favorece plural: “Os Estados Unidos anunciaram”.
- título de obra pode ser tomado como unidade: “Memórias Póstumas... é uma obra”.
- não decida apenas pela terminação gráfica.

## 8. Impessoais

- **haver** = existir/ocorrer/tempo → singular.
- locução: “deve haver”, “pode haver”.
- **fazer** = tempo/clima → singular.
- **existir, ocorrer, surgir** → concordam com o sujeito.
- **haja vista** → invariável no padrão editorial.
- hora/distância com **ser** → acompanha a expressão numérica.

## 9. Construções com `se`

| Estrutura | Regra |
| --- | --- |
| VTD/VTDI + se apassivador | verbo com sujeito paciente |
| VI/VTI/VL + se indeterminador | 3ª pessoa singular |
| “Publicaram-se os editais” | passiva |
| “Precisa-se de analistas” | indeterminação |
| “Devem-se revisar os atos” | atos devem ser revisados |
| “Pode-se recorrer” | singular; VTI |

## 10. Infinitivo

- sujeito próprio expresso → flexão frequentemente necessária.
- locução verbal → infinitivo não se flexiona.
- mesmo sujeito fora de locução → pode haver facultatividade.
- “Aos gestores cabe revisar” → sujeito de *cabe* é a oração.
- “Antes de os gestores revisarem” → sujeito expresso do infinitivo.

## 11. Cadeias passivas

> auxiliar + ser + particípio/predicativo

- “A ata e o parecer **tinham sido aprovados**.”
- “A ata e a certidão foram consideradas **válidas**.”
- Verifique auxiliar, particípio e predicativo separadamente.

## 12. Concordância nominal

- um nome → determinantes e adjetivos com o nome.
- adjetivo anteposto → proximidade, em regra.
- adjetivo posposto → plural abrangente ou proximidade, conforme sentido.
- predicativo de sujeito composto → plural.
- gêneros diferentes no plural abrangente → masculino plural.

## 13. Formas recorrentes

| Forma | Regra |
| --- | --- |
| anexo, incluso, apenso | variáveis |
| em anexo | invariável |
| mesmo/próprio reflexivos | variáveis |
| mesmo inclusivo/concessivo | invariável |
| quite | variável |
| obrigado | concorda com quem agradece |
| só = sozinho | variável |
| só = somente | invariável |
| meio = metade | variável |
| meio = um pouco | invariável |
| bastante = suficientes | variável |
| bastante = muito | invariável |
| menos | invariável |
| dado/visto participiais | variáveis |
| salvo/exceto preposicionais | invariáveis |
| alerta | depende do emprego |
| leso | varia no composto |

## 14. `Possível` e expressões predicativas

- “as medidas mais rápidas **possíveis**”.
- “medidas **o mais rápidas possível**”.
- sem determinante: “É necessário paciência”.
- com determinante: “É necessária esta paciência”.
- sujeito oracional: “É necessário que todos participem”.

## 15. Cores

- cor adjetiva simples → varia.
- substantivo usado como cor → tende a ser invariável.
- composto de adjetivos → tradição flexiona o último.
- consulte dicionário para compostos lexicalizados.

## 16. Pegadinhas

| Afirmação | Julgamento |
| --- | --- |
| plural próximo controla o verbo | errado |
| sujeito oracional leva verbo ao plural | errado |
| cada um dos admite plural livre | errado |
| nome próprio plural sempre exige plural | errado |
| haja vista varia | errado no padrão adotado |
| alerta é sempre invariável | errado |
| todo particípio concorda com objeto | errado |
| auxiliar correto garante particípio correto | errado |
| `haver` existencial tem sujeito | errado |
| reescrita correta preserva sempre o foco | errado |

## 17. Método de prova

1. Localize a forma flexionada.
2. Classifique sua função.
3. Ache o controlador e o núcleo.
4. Verifique impessoalidade, passiva ou locução.
5. Examine variantes documentadas.
6. Propague alterações em reescritas.
7. Teste correção e sentido separadamente.
