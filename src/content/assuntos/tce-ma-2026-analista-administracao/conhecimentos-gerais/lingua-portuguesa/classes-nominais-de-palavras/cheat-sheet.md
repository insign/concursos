# Classes nominais de palavras

## Regra de ouro

- Classe gramatical, sentido e função sintática são planos diferentes.
- Analise a palavra na oração: a mesma forma pode mudar de classe conforme o emprego.
- Classes deste assunto: substantivo, artigo, adjetivo, numeral e pronome.

## Substantivo

| Critério | Classificações |
| --- | --- |
| Extensão | comum / próprio |
| Natureza | concreto / abstrato |
| Formação | primitivo / derivado |
| Estrutura | simples / composto |
| Coletividade | coletivo |
| Gênero | masculino/feminino; comum de dois; sobrecomum; epiceno |

- Concreto não é sinônimo de palpável: `anjo` é tradicionalmente concreto.
- Coletivo tem forma singular: `o elenco chegou`.
- Comum de dois: `o/a estudante`.
- Sobrecomum: `a criança`, `a vítima`, `o cônjuge`.
- Epiceno: animal com uma forma gramatical: `a cobra macho/fêmea`.
- Diminutivo/aumentativo podem expressar afeto, ironia ou depreciação, não só tamanho.
- Substantivação: `o belo`, `o não`, `um porquê`.

## Artigo

| Tipo | Formas | Uso |
| --- | --- | --- |
| Definido | `o, a, os, as` | referente identificado/delimitado |
| Indefinido | `um, uma, uns, umas` | referente não identificado ou apresentado |

- `um servidor`: artigo; `um, não dois`: numeral.
- `o relatório`: artigo; `eu o li`: pronome oblíquo.
- Contrações: `do = de + o`; `na = em + a`; `ao = a + o`.
- Artigo pode substantivar: `o não`, `um porquê`.
- Artigo definido, indefinido e ausência de artigo podem mudar a referência.

## Adjetivo

- Caracteriza ou relaciona o substantivo: `decisão justa`, `controle financeiro`.
- Uniforme: `servidor/servidora competente`.
- Biforme: `servidor atento` / `servidora atenta`.
- Gentílico: `maranhense`, `brasileiro`.
- Locução adjetiva: `amor de mãe`, `decisão de governo`.
- Substantivação: `o brasileiro` x `cidadão brasileiro`.

| Posição | Mudança frequente de sentido |
| --- | --- |
| `grande homem` / `homem grande` | notável / de grande porte |
| `velho amigo` / `amigo velho` | de longa data / idoso |
| `pobre homem` / `homem pobre` | digno de pena / sem recursos |

| Grau | Modelo |
| --- | --- |
| Comparativo de igualdade | `tão útil quanto` |
| Superioridade/inferioridade | `mais/menos útil que` |
| Sintético | `melhor`, `pior`, `maior`, `menor` |
| Superlativo absoluto | `muito útil`, `utilíssimo` |
| Superlativo relativo | `o mais útil` |

## Numeral

| Classe | Exemplos |
| --- | --- |
| Cardinal | `um`, `dois`, `cem`, `mil` |
| Ordinal | `primeiro`, `vigésima` |
| Multiplicativo | `dobro`, `triplo` |
| Fracionário | `meio`, `terço`, `quartos` |

- `duzentas páginas`; `primeira etapa`.
- `dois mil candidatos`, sem plural em `mil`.
- `dois milhões`, com plural em `milhão`.
- `meia hora`: numeral; `meio cansada`: advérbio invariável.
- `ambos/ambas` = os dois/as duas.

## Pronome

| Tipo | Exemplos e alerta |
| --- | --- |
| Pessoal reto | `eu, tu, ele`; tipicamente sujeito |
| Oblíquo átono | `me, te, o, a, lhe`; liga-se ao verbo |
| Oblíquo tônico | `mim, ti, si`; normalmente com preposição |
| Tratamento | concordância na 3ª pessoa: `Vossa Excelência decidiu` |
| Possessivo | concorda com possuído: `minhas decisões` |
| Demonstrativo | `este, esse, aquele`; `isto, isso, aquilo` são neutros |
| Relativo | `que, quem, cujo, onde, o qual` |
| Indefinido | `alguém, ninguém, qualquer`; plural: `quaisquer` |
| Interrogativo | `que, quem, qual, quanto` |

- `para eu redigir`: `eu` é sujeito do infinitivo.
- `para mim`: `mim` não pratica a ação.
- `seu/sua` pode ser ambíguo; `dele/dela` pode esclarecer.
- Modelo tradicional: `este` pode anunciar; `esse` pode retomar.
- `isto, isso, aquilo` não variam nem acompanham substantivo expresso.
- Regência define preposição: `o assunto de que tratei`; `a pessoa a quem me referi`.
- `cujo` indica posse, concorda com o termo seguinte e não admite artigo: `cujas obras`.
- `onde` retoma lugar; em outros casos, use `em que` ou `no qual`, conforme a regência.

## Roteiro de prova

1. Leia toda a oração.
2. Identifique o que a palavra faz: nomeia, determina, qualifica, quantifica ou retoma.
3. Diferencie forma igual em usos diferentes: `um`, `o/a`, `meio`, `brasileiro`.
4. Confira gênero, número, grau ou pessoa.
5. Em reescritas, preserve referência, posse, quantidade e sentido.
